New [Yandex Alice](https://yandex.ru/dev/dialogs/alice/) skill.

## Usage

### Flags

- a: address and port to run server (default `:8080`)
- l: log level (default `info`)

### Envs

- RUN_ADDR: address and port to run server
- LOG_LEVEL: log level
